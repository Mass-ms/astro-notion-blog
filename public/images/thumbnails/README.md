# サムネイル画像フォールバック

このディレクトリには、Notionでサムネイル画像が設定されていない記事用のフォールバック画像を配置します。

## 必要な画像

### 必須
- **default.jpg** - タグがない、または対応する画像がない場合に使用される汎用画像

### オプション（タグ別）
- **dtm.jpg** - 「DTM」タグの記事用
- **news.jpg** - 「お知らせ」タグの記事用
- **tips.jpg** - 「小ネタ」タグの記事用
- **dev.jpg** - 「開発ログ」タグの記事用

## 推奨仕様

- **サイズ**: 1200x630px (OGP標準)
- **フォーマット**: JPEG、PNG、WebP
- **ファイルサイズ**: 500KB以下推奨
